# sqlite_manager

## 项目简介

SQLite 在线管理器是一个基于 Web 的 SQLite 数据库管理工具，允许用户在浏览器中直接操作 SQLite 数据库文件，无需安装任何软件。所有操作均在本地完成，数据不会上传到服务器，保护用户数据隐私。

## 主要功能

- 📁 **文件操作**
  - 上传 SQLite 文件 (.db/.sqlite/.sqlite3)
  - 新建数据库
  - 保存数据库文件到本地
  - 支持拖放文件上传

- 💻 **SQL 编辑与执行**
  - 语法高亮的 SQL 编辑器
  - 执行 SQL 语句
  - 查看查询结果
  - 清空编辑器内容

- 📊 **数据表管理**
  - 查看数据库中的所有表
  - 浏览表数据
  - 编辑现有数据
  - 添加新数据

- 📱 **响应式设计**
  - 支持桌面端和移动端访问
  - 自适应不同屏幕尺寸

## 技术栈

- **前端框架**：纯 JavaScript (ES6+)
- **样式框架**：Tailwind CSS v3
- **图标库**：Font Awesome 4.7
- **SQLite 引擎**：SQL.js (WebAssembly)
- **字体**：自定义字体 (xst.woff2)

## 快速开始

### 方法一：直接使用

1. 下载项目文件到本地
2. 用浏览器打开 `sqlite.html` 文件即可使用

### 方法二：通过本地服务器运行

```bash
# 使用 Python 启动本地服务器
python -m http.server 8000

# 或者使用 Node.js (需要安装 http-server)
npx http-server
```

然后在浏览器中访问 `http://localhost:8000/sqlite.html`

## 使用指南

### 1. 上传或新建数据库

- **上传现有文件**：点击「上传文件」按钮，选择你的 SQLite 数据库文件
- **拖放上传**：直接将 SQLite 文件拖放到页面上
- **新建数据库**：点击「新建数据库」按钮创建一个空的数据库

### 2. 编写和执行 SQL

1. 在 SQL 编辑器中输入 SQL 语句
2. 点击「执行 SQL」按钮运行语句
3. 查询结果会显示在下方的结果区域

### 3. 管理表数据

- **查看表**：在数据表列表中点击表名查看表结构和数据
- **编辑数据**：点击结果表中的编辑按钮修改现有数据
- **添加数据**：点击「添加数据」按钮插入新记录

### 4. 保存数据库

编辑完成后，点击「保存文件」按钮将修改后的数据库保存到本地。

## 注意事项

1. 所有数据处理均在浏览器中完成，不会上传到任何服务器
2. 大文件处理可能会导致浏览器卡顿，建议处理小于 100MB 的数据库文件
3. 浏览器刷新或关闭后，未保存的数据将会丢失
4. 依赖网络加载 SQL.js 引擎，首次使用需要网络连接
